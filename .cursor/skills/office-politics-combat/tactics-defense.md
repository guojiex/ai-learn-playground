# Tactics：防守型剧本

**防守 = 在不升级冲突的前提下，把对方甩过来的东西原路退回、不接锅、不被框死。**

每个招式给出：**使用场景 / 原理 / 话术模板 / 搭配的钩子 / 代价**。

---

## 1. 拒锅（Return to Sender）

**原理：** 锅是具体的，不接锅的关键是**指出这个锅不属于我**，并给出正确的归属方。

**三步法：**

1. **确认事实**：重复对方说的具体事项（避免被后续否认）
2. **指出归属**：按职责 / 流程 / 已有文档，这事应由谁负责
3. **给建议**：建议把事情推给正确的人，表现得不是"推"，是"帮对方理顺"

**话术模板（标准档）：**

> 我确认一下 <事情>：你是希望我 <具体动作> 对吗？
> 不过按现在 <流程 / 文档 / 分工，最好带链接>，这块应该是 <正确归属方> 负责。建议我们直接拉 <对方> 看一下这个事，会更高效。我这边可以提供 <辅助动作>。

**搭配钩子：** 责任归位钩子、文档引用钩子。

**代价：** 对方会觉得你"不配合"。**必须配合做辅助动作**（让渡一点来换大头不接），避免被定义为"阻塞"。

---

## 2. 反问归因（Reverse the Burden）

**原理：** 对方扔过来一个模糊指控 / 诉求，不辩解、不自证，直接把举证责任丢回去。

**关键原则：** **永远不自证。** 自证就是默认指控有理。

**话术模板：**

| 对方指控 | 不要说 | 要说 |
|---------|-------|------|
| "你们团队效率不高" | "我们其实很忙……" | "具体指哪件事？我看一下对应的数据。" |
| "你不够 AI 思维" | "我也学了 AI……" | "具体是哪件事让你这么判断？期望动作是？" |
| "别的团队都能做" | "我们也能做，但……" | "能同步一下别的团队是怎么做的吗？我对齐一下。" |
| "这个简单" | "其实不简单……" | "简单的话方便给个实现思路吗？我可能理解得复杂了。" |
| "你情绪很大" | "我没有情绪" | "我情绪稳定。回到刚才的议题：X。" |

**搭配钩子：** 衡量标准钩子（见 [hook-patterns.md](hook-patterns.md)）。

**代价：** 反问 2-3 次后对方如果还没具体化，你的不满就合理化了，对方反而尴尬。

---

## 3. 留白 / 不立刻回复

**原理：** 对方在 IM 要求你立刻表态时，**沉默本身就是防守**。给自己时间，把对话搬到更有利的渠道。

**操作：**

- IM @ 你催回复 → 先回一句"收到，我想一下再同步"，**然后离开 1-4 小时**
- 群里被突然点名 → "收到，我会后整理一下再回 @XX"（把场合从群改成更私下的文档）
- 会议上被突然袭击 → "这个问题我需要看一下数据，会后给你完整答复"（**永远不要在没准备时当场承诺**）

**话术模板：**

> 收到。这个事我想清楚再回你，下班前给答复。

**搭配钩子：** 无（留白本身是反钩子动作）。

**代价：** 对方会觉得"拖延"。**承诺的时间必须守**，且到点了主动回，否则失分。

---

## 4. 合规化 / 走流程

**原理：** 对方想跳过流程让你吃亏时，把流程搬出来。流程是合规武器。

**用法：**

- 对方想让你加班赶工 → 走加班申请流程
- 对方想让你接一个不该接的活 → 走需求评审流程
- 对方想让你承担一个没书面化的 KPI → 要求走 OKR 流程对齐
- 对方想让你不要周报 / 日报写某事 → 拒绝，周报按实际写

**话术模板（标准档）：**

> 这个事我也想尽快支持。按我们 <流程 / RACI / 规范> 这块需要 <具体步骤>。我可以先把 <我这边能做的动作> 启动，流程那一步 <哪位同学> 能帮忙推一下？

**搭配钩子：** 文档引用钩子。

**代价：** 几乎无代价，最多显得"死板"。**反而让老板知道你不是可以随便揉的软柿子**。

---

## 5. 延迟放行（Conditional Yes）

**原理：** 不直接拒绝（会被贴"不配合"标签），而是同意——**但有条件**。

**用法：**

- "可以做，但需要 <前置条件>"
- "可以推进，但 deadline 需要 <更长时间>"
- "可以接，但 <另一项工作> 需要降优先级"

**话术模板（标准档）：**

> 这个事我可以接。我这边本周还有 A / B / C 在跑，按优先级大概能在 <具体时间> 出初稿。如果要更快，A / B / C 里哪个可以降优先级？你拍一下？

**搭配钩子：** 优先级确认钩子（把资源分配的决策权逼回给对方，出问题是对方的锅）。

**代价：** 对方可能真的给你加任务。**一定要在同一句里把代价说清楚**，不能只说"可以做"。

---

## 6. 一致性挑战（Consistency Trap）

**原理：** 对方说过的话 / 写过的文档 / 公开的承诺，本身就是约束他的工具。把他过去的话翻出来对比现在的要求。

**用法：**

- 对方今天要 X，但上周说过反对 X → "我理解上周对齐的是 Y，这次调整是因为什么新情况？"
- 对方对 A 和 B 两个下属施加不同标准 → "我看 A 那个项目也是类似情况，当时的处理是 C。这次我这边建议也走 C，避免后续同学不好对齐。"
- 对方公开宣称"我们鼓励讨论"但私下打压你 → "按之前团队共识，遇到问题鼓励提出来，所以我想把 X 再理一下。"

**话术模板（温和档）：**

> 我先理解一下哈，之前在 <时间 / 场合> 你说过 <对方的原话>，这次 <新的要求> 是基于什么新的考虑？我更新一下认知。

**搭配钩子：** 书面化钩子（把对方的新立场也落成文档，下次可再用）。

**代价：** 对方会意识到你有记忆、有记录。**用词必须是"更新认知"这类中性词**，不能显得"抓把柄"。

## 7. 装傻 / 战略性纯真（Strategic Naivety）

**原理：** 对方用暗示 / 潜台词让你做脏活（比如"你懂我意思吧？"），**装作听不懂**，逼对方把脏话明说，他就不敢说了。

**用法：**

- Boss 暗示你去打压同事 → "抱歉我没完全理解，你希望我具体做什么动作？是在周会上提出来，还是和他私下聊？"
- Boss 暗示"你懂怎么让这个项目延期吧" → "我目前看是能按时交的，你是担心哪个具体风险？"
- 要求你做灰色操作 → "这个步骤我想确认一下合规性，能同步给 <法务 / 合规 / HR> 对一下吗？"

**话术模板：**

> 抱歉我没完全 get 到。你期望的具体动作是什么？A 还是 B？我按具体动作来做。

**搭配钩子：** 责任归位钩子（把潜台词变成明确指令，对方要么明说要么放弃）。

**代价：** 对方可能会觉得你"情商低"。**这正是战略性装傻的价值**——用"情商低"换"不背锅"。

---

## 8. 最小接触（Minimum Engagement）

**原理：** 对于反复骚扰 / 低价值对话，不给能量、不回情绪、只回必要事实。

**用法：**

- 对方情绪化长篇指责 → 只回一句事实："我看了你刚才的消息。按 <文档>，这块责任是 <对方>，我会 <具体动作>。"
- 对方反复问"你觉得呢" 想让你表态 → "我按 <已经对齐的> 来推。有新决定同步到群里。"
- 对方试图把私人关系带入工作 → "这个事我们走 <正式渠道>。"

**原则：**

- 不回复对方的情绪
- 不评价对方的人
- 只回复具体事项 + 具体动作

**代价：** 几乎无。**但要和拉盟友策略配合用**，否则容易显得孤僻。

---

## 9. 止损降速（Damage Control）

**原理：** 当局面已经失控（被当众骂、被踢群、PIP 威胁），不争一时，先止损，**保护好证据和下一步路**。

**操作顺序：**

1. **止血**：当场不反驳（反驳会让对方加码），记录时间地点在场人
2. **截图 / 保存证据**：IM / 邮件 / 会议录音（如合规）立即备份到个人设备
3. **找信任的同事复盘**：听第三方视角
4. **HR / 升级**（仅在证据充分 + 有后路时）

**话术模板（当场被公开指责）：**

> 我听到了。具体细节我会后整理一份回复，今天下班前发给你 / 同步到群里。

**搭配钩子：** 无（止损阶段不埋钩子，先稳住局面）。

**代价：** 当场吃哑巴亏。**但把"正式回复"留到自己准备充分时，往往能反击得更漂亮**。

---

## 防守型原则速记

1. **不自证**。所有防守第一步都是把举证责任丢回去。
2. **不接情绪**。只回事实和动作。
3. **不当场承诺**。任何新任务 / 新要求都要留个"我想一下"的空间。
4. **守流程**。流程是最合规的武器。
5. **保证据**。每次被针对的当天就把聊天记录备份到个人设备。
6. **不孤立防守**。防守时同步拉盟友、积累证据，为未来的进攻或升级做准备。
